[![Build Status](https://travis-ci.org/linas-p/Biser.svg?branch=master)](https://travis-ci.org/linas-p/Biser)

Requirements: c++11 i.e g++4.8/clang3.3 or better


To compile type

```sh
make
```





----------------------------------


To test raw code:

```sh
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./BiserLikeModel

```

To change params change file
```sh
$ config/params.json
```
